# Ascend-Forum


Ascend Forum is a web-based discussion platform focused on self-improvement and aesthetics.  
The goal of the project is to provide a clean, modern environment where users can create accounts, participate in discussions, and share knowledge related to looks, fitness, style, and personal development.


Main Features

- Authentication using ASP.NET Core Identity.
- Create, view and manage forum categories.
- Admin panel.
- Post and participate in discussions.
- Responsive design with Bootstrap.
- Clean UI.


Technologies used

- ASP.NET Core
- .NET 10
- Razor Pages / MVC
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- Bootstrap 5
- HTML, CSS

Setup instructions:

1. Clone the repository.

2. Open the solution file in Visual Studio.

3. Update the connection string in 'appsettings/json'. Run 'Update-Database'.

4. Run the application.
